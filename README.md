Tekstile - Baskı Yönetim Sistemi
Bu proje, tekstil sektörüne yönelik olarak geliştirilmiş kapsamlı bir baskı yönetim uygulamasıdır.
C# Windows Forms ile geliştirilmiş olup Entity Framework Core ile katmanlı mimaride inşa edilmiştir.
Uygulama; boya yönetimi, desen-boya ilişkileri, stok takibi, muhasebe modülü (maaş hesaplama gibi), sipariş yönetimi ve kullanıcı yetkilendirme gibi birçok modül içermektedir.
Gerçek bir işletme senaryosuna uygun olarak geliştirilmiş ve tüm işlemler veri tabanı üzerinden yönetilmektedir.

Kullanılan Teknolojiler:

C# (.NET)

Windows Forms

Entity Framework Core (Code First)

Katmanlı Mimari

SQL Server

Dependency Injection

DTO, Validation, Logging
